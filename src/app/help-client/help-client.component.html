Оффициальный клиент OpenVPN можно скачать с этой страницы. Последнюю версию (на момент создания проекта) можно скачать по этой прямой ссылке.

После установки, надо загрузить ключ с настройками подключения в установленный клиент. Подробную инструкцию (в картинках) по установке и загрузке файла ключа можно посмотреть, к примеру, здесь

В основных linux системах, openvpn пакет установлен, и vpn соединяния управляются сетевым менеджером. Обычно, требуется установить дополнительный пакет для удобного создания vpn подключений из *.ovpn файлов.

Например в linux ubuntu/debian:
apt-get install network-manager-openvpn-gnome
После установке, выберите пункт "редактировать соединения" в меню сетевого менеджера, и добавьте новое из скачаного ovpn файла ключа.

Для MacOS требуется установить программу TunnelBlick. Инструкции по загрузке ovpn файла ключа в установленный tunelblik можно посмотреть тут.
